# CJ NET Printing System

## Goal
Build a simple QR-based file submission system for a printing shop.

## Rules
- No authentication
- No overengineering
- Mobile-first UI
- Staff must manually confirm prints
- Utility app first, not SaaS
- Keep screens direct, quiet, and easy to scan
- Avoid technical or intimidating labels unless they are required for the task

## Features
- Landing page
- Upload system
- Admin dashboard

## Tech
- Next.js
- TypeScript
- File system storage
